# Aparecida nutricionista 


Resumo do que foi utilizado para criação da página web;

- filtragem de nomes 
- remoção de nomes com duplo click
- adicionar paciente a tabela 
- validação dos dados
- busca de pacientes utilizando ajax
- calcula imc dos pacientes
